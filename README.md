# 맛집리뷰 API
### 🌱 개발 목적 및 개요
맛집관리, 리뷰 및 별점 관리를 통한 토이 프로젝트로 공부목적

- REST API 이해
- SPRING BOOT 공부 - 어노테이션, H2 Database 등 이해
- JPA 공부 - Repository, QueryDSL, Pageable, Slice 등 이해 
- Docker 및 AWS 배포를 통한 공부 및 기록

### 사용한 프레임웍, 라이브러리 및 툴
- IntelliJ Ultimate
- Postman
- Docker
- JAVA 17
- Lombok
- Spring Boot
- AWS EC2
